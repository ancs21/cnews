## CNews

Cralwer tiêu đề bài bào trên các web site tin tức Việt Nam

### Cài đặt
- Tạo file `.env` chưa thống số MongoDB
```
MONGODB=
DBNAME=
```

- Run
```
node index.js [name_news]
```
Ví dụ
```
node index.js nlp # Báo Người lao động
```

### Stack
- Nodejs v10.10
- MongoDB
